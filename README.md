# ASTRA-CLIENT

ASTRA-CLIENT is a custom Gorilla Tag client made for experimenting, learning, and testing new features in Unity and BepInEx. It’s a work-in-progress project where features get added, changed, and updated often.

## Features
- Custom UI menus  
- Experimental movement tools  
- Gameplay tweaks  
- Utility tools for modding  


## Requirements
- BepInEx 5.x  


## Installation
   1. Donload the '.dll' from the newest release **Download link: https://github.com/ASTRA228b/ASTRA-CLIENT/releases**
   2. Open Steam click on 'Gorilla Tag' then click on the gear thingy in there click on manage then browse local files then go to the `BepInEx/plugins` folder
   3. Drag the '.dll' file into your `BepInEx/plugins` folder then start your game an injoy
